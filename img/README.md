# Portada (imagen)

Para que la portada del libro use una imagen:

- Coloca una imagen en `img/portada.jpg` (recomendado) o `img/portada.png`.
- Alternativa (PDF): `img/portada.pdf`.
- Si quieres otra ruta/nombre, define `\LibroPortadaArchivo{img/mi_portada.jpg}` en `tex/metadatos.tex`.

Notas prácticas (Amazon/KDP):
- Mantén proporción 6x9 (misma proporción que el interior por defecto).
- Para eBook, la portada normalmente es un JPEG/PNG y se valida en el previewer de Kindle/KDP.
- La portada del impreso suele entregarse aparte (portada completa + lomo), según la plantilla de KDP.
